# Product Overview

Construction ERP mobile application for workforce management and field operations.

## Core Features

- Worker task management and tracking
- Supervisor oversight and daily progress reporting
- Driver/fleet management for transportation
- Attendance monitoring with geofence validation
- Real-time notifications and alerts
- Photo documentation and progress updates
- Material/tool requests and approvals
- Leave and payment request workflows

## User Roles

- Workers: Field employees completing assigned tasks
- Supervisors: Team leads monitoring progress and approving requests
- Drivers: Fleet operators managing transport tasks
- Company Users: Administrative staff with system-wide access

## Key Workflows

- Task assignment and execution with geofence validation
- Daily attendance check-in/check-out
- Progress reporting with photo evidence
- Request submission and approval chains
- Real-time location tracking for compliance
