## â“ COMMON QUESTIONS

**Q: Can I submit multiple reports per day?**  
A: No, one DPR per project per day.

**Q: What if I forget to submit?**  
A: Submit the next day with the correct date.

**Q: Can I edit after submission?**  
A: No, only Admin can edit submitted reports.

**Q: Are all fields required?**  
A: No, many fields are optional. Required fields are marked with *.

**Q: What happens to my photos?**  
A: Photos are stored centrally and linked to your report.

**Q: Can I work offline?**  
A: Drafts can be saved offline and submitted when online.
